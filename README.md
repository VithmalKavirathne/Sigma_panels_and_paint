# Sigma Panels & Paint Website

## Project Details
- **Project Name**: Sigma Panels & Paint Website

## Important Development Rule
> [!IMPORTANT]
> The folder `stitch_sigma_digital_showroom/` is **read-only** and is used **only** as a design reference library.
